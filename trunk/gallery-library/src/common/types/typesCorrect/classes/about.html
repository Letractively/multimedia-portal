коды ошибок исправление проводится только если код > 0
код название описание
0 совпадают значение соответствует типу
-1 полное не соответствие значение не соответствует типу
-2 адрес почты адрес почты был введен не верно
-3 integer число введено не правильно
-4 phone телефон введен не правильно
-5 login login имеет недопустимые символы
-6 password пароль имеет недопустимые символы
-7 float float имеет недопустимые символы
-8 url url составлен не правильно
-9 url неправильный формат даты дата имеет формат ГГГГ-ММ-ДД ЧЧ:мм:сс
-10 intArray ошибка в массиве цифр(integer)
1 исправляемая ошибка введенное значение может быть исправлено до подходящего автоматически
2 исправляемая ошибка в тексте надо замениль &, <, >, '
3 исправляемая ошибка в сгенерируемом коде подается любая строка и вегда получаем код ошибки надо сгенерировать код разрядностью 36
4 исправляемая ошибка в float разделитель между целой и дробной частью должен быть . однако найдена , (можно заменить)
